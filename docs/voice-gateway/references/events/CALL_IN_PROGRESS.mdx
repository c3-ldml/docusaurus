# CALL_IN_PROGRESS

<a href="Added"><img src="https://img.shields.io/badge/Added_in-v4.90-blue" alt="Version badge" /></a>

The call is in progress once the call has been successfully established.

To capture this event, follow these steps:

1. In your voice Flow, add a [Question](../../../ai/build/node-reference/basic/question.mdx),
   [Optional Question](../../../ai/build/node-reference/basic/optional-question.mdx),
   or [Wait For Input](../../../ai/build/node-reference/logic/wait-for-input.mdx) Node before the [Lookup](../../../ai/build/node-reference/logic/lookup.mdx) Node
   where the `CALL_IN_PROGRESS` event is selected.
2. In the [Voice Gateway](../../../ai/deploy/endpoint-reference/voice-gateway.mdx) Endpoint, activate the **Enable Call Event: Call in Progress** setting.
3. Navigate to the **Call Events** section and enable the **Call in Progress** option.
