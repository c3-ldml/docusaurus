# Voice Gateway Self-Service Portal

The Cognigy Voice Gateway Self-Service Portal is a web-based interface that allows users to manage their accounts, settings, and preferences for Cognigy Voice Gateway.

The Web App has the following entities:

- [Users](users.mdx)
- [Accounts](accounts.mdx)
- [Applications](applications.mdx)
- [Recent Calls](recent-calls.mdx)
- [Alerts](alerts.mdx)
- [Carriers](carriers.mdx)
- [Speech](speech-services.mdx)
- [Phone Numbers](phone-numbers.mdx)
- [Outbound Call Routing](outbound-call-routing.mdx)


## How to start

If you use the Cognigy Voice Gateway Self-Service Portal for the first time, ensure that you have completed the steps in the [Getting Started guide](../getting-started.mdx). Once you have completed the guide, proceed with the following steps:

1. Identify the [user scope](users.mdx) you have or add a new user. The operations that you can perform in Voice Gateway are based on your user scope, such as Admin, Service Provider, or Account.
2. Create an [account](accounts.mdx) or use an existing one.
3. Create an [application](applications.mdx) or use an existing one.
4. Add a [new carrier](carriers.mdx) or use an existing one.
5. Add a [phone number](phone-numbers.mdx) or use an existing one.
6. Add [speech provider credentials](speech-services.mdx) or use existing ones.

Additionally, you can:

- Check [recent calls](recent-calls.mdx) and view all information about the call.
- Check [alerts](alerts.mdx) if something went wrong with call configuration.
- Configure [outbound call routing](outbound-call-routing.mdx).
