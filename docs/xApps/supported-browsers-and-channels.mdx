# Supported Browsers and Channels

Cognigy xApps are compatible with the browsers and channels listed below.

## Supported Browsers

The following browsers are supported by xApps:

| Browser  | Versions                                      |
|----------|-----------------------------------------------|
| Firefox  | The newest version and two previous versions. |
| Chrome   | The newest version and two previous versions. |
| Chromium | The newest version and two previous versions. |
| Safari   | The newest version and two previous versions. |
| Edge     | The newest version and two previous versions. |

## Supported Channels

The following channels are supported by xApps:

- [Voice Gateway](../ai/deploy/endpoint-reference/voice-gateway.mdx)
- [AudioCodes](../ai/deploy/endpoint-reference/audiocodes.mdx)[^*]
- [Facebook (Meta) Messenger](../ai/deploy/endpoint-reference/facebook-messenger.mdx)
- [LINE Messenger](../ai/deploy/endpoint-reference/line-messenger.mdx)
- [Azure Bot Services](../ai/deploy/endpoint-reference/azure-bot-services.mdx)
- [RingCentral Engage](../ai/deploy/endpoint-reference/ringcentral-engage.mdx)
- [Slack](../ai/deploy/endpoint-reference/slack.mdx)
- [Socket](../ai/deploy/endpoint-reference/socketio.mdx)
- [Sunshine Conversation](../ai/deploy/endpoint-reference/sunshine-conversations.mdx)
- [WhatsApp](../ai/deploy/endpoint-reference/whatsapp.mdx)
- [Webchat v2](../webchat/v2/overview.mdx)
- [Webchat v3](../webchat/v3/overview.mdx)[^**]
- [Webhook](../ai/deploy/endpoint-reference/webhook.mdx)
- [REST Endpoint](../ai/deploy/endpoint-reference/rest.mdx)


[^*]: For this channel, Async Mode needs to be activated in the Endpoint settings.
[^**]: Webchat v3 is the only channel that supports xApp pages embedded within the channel interface. For more information, read the [xApp Overlay](build/overlay.mdx) article.
