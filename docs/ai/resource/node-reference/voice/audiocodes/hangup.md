## Description

The Hangup Node will instruct AudioCodes to end the call.

| Parameter | Type          | Description                                              |
|-----------|---------------|----------------------------------------------------------|
| Reasons   | CognigyScript | The reason for hanging up. Will show in AudioCodes logs. |
