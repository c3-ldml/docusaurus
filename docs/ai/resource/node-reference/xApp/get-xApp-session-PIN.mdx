# xApp: Get Session PIN

<img src={require('../../../../../static/img/_assets/ai/resource/node-reference/xApp/get-xApp-session-PIN.png').default} width="50%" />

## Description

This Node generates a new xApp session PIN. You can access it using the [xApp: Get Session PIN](../../../../xApps/tokens.mdx#xapp-session-pin) token or via `input.apps.session.pin`.

The PIN code is only valid for 5 minutes.

With this PIN, users can log in via an xApp PIN webpage.

### xApp PIN Page URL

Share the xApp PIN Page URL with users, where they can enter the PIN. The URL can be sent in the Flow with an [xApp PIN Page URL](../../../../xApps/tokens.mdx#xapp-pin-page-url) Cognigy token.
