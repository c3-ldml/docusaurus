# Resource Pages

Resource pages are pages where you can view and manage resources to build AI Agents, for example, Flows, Knowledge Stores, Endpoints, and many others. You can access resource pages in the [left-side menu](project-page.mdx) of the **Project** page.

The following image provides an overview of the Flows page:
